moldinam
========

Molecular dinamic project

Requirements
--------
C++ compiler with C++11 support<br />
Boost (temporary)

How to build
--------

**Linux:**  
`cd moldinam`  
`mkdir build`  
`cd build`  
`cmake ..`  
`make install`  

or you can build it with python script automatically

`cd moldinam`  
`export MD_ROOT=<path to moldinam root>`  
`__env/mkbuild.py` 

**Windows:** <br />
Not supported yet
